pyshgp.push.interpreter module¶

The interpreter module defines the PushInterpreter class.

A PushInterpreter is capable of running Push programs.

run(program: pyshgp.push.atoms.CodeBlock, inputs: Sequence, output_types: Sequence[str], verbose: bool = False)[source]¶

Run a Push program given some inputs and desired output PushTypes.

The general flow of this function is:
  1. Create a new push state
  2. Load the program and inputs.
  3. If the exec stack is empty, return the outputs.
  4. Else, pop the exec stack and process the atom.
  5. Return to step 3.”
Parameters:
  • program (CodeBlock) – Program to run.
  • inputs (Sequence) – A sequence of values to use as inputs to the push program.
  • output_types (Sequence[Union[PushType, type, str]]) – A secence of values that denote the Pushtypes of the expected outputs of the push program.
  • verbose (bool) – If true, program execution steps will be printed. Default False.
Returns:

A sequence of values pulled from the final push state. May contain pyshgp.utils.Token.no_stack_item if needed stacks are empty.
Return type:

Sequence
